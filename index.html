        లిపి & భాష
        🔤
        లిపి ఎంపిక (Script)
        పాఠం టాబ్ పై తె / En / सं బటన్లతో లిపి మారుస్తారు. Settings లో కూడా మార్చవచ్చు. మీరు ఎంచుకున్న లిపి అన్ని వ్యాసాలకూ వర్తిస్తుంది.
        🎨
        లిపి రంగు vs అర్థ రంగు
        నేరుస్తున్న రంగు (మారూన్/accent) = లిపి పాఠ్యం — శ్లోక పాదాలు, నామాలు. సాధారణ రంగు (నలుపు/తెలుపు) = అర్థాలు, లేబుళ్లు, UI. ఈ తేడా వల్ల ఏ పాఠ్యం లిపి మారినప్పుడు మారుతుందో తెలుస్తుంది.
        🌐
        UI భాష vs అర్థ భాష
        UI భాష — బటన్లు, శీర్షికలు తెలుగు లేదా English లో. అర్థ భాష — శ్లోక అర్థాలు ఏ భాషలో చూపించాలో ఎంచుకోండి: English (En), తెలుగు (తె), లేదా Sanskrit (सं).
        పాఠం (Reader)
        📚
        గ్రంథం ఎంపిక
        పాఠం టాబ్ పై dropdown లో శ్రీమద్భగవద్గీతా లేదా శ్రీవిష్ణుసహస్రనామమ్ ఎంచుకోండి. మీ ఎంపిక తదుపరి సారి కూడా గుర్తుంటుంది.
        📖
        అధ్యాయ ఎంపిక
        ఒకటి లేదా అనేక అధ్యాయ బటన్లు నొక్కండి. బహు అధ్యాయాలు ఎంచుకుంటే Random నొక్కినప్పుడు ఆ అన్నింటి నుండి శ్లోకం వస్తుంది. గ్రంథాన్ని మార్చినప్పుడు ఎంపిక reset అవుతుంది.
        ☀️
        నేటి శ్లోకం (Today's Verse)
        ☀️ బటన్ నొక్కితే నేటి తేదీకి నిర్ణయించిన నిర్దిష్ట శ్లోకం వస్తుంది — 700 శ్లోకాలలో రోజువారీ మారుతుంది, ప్రపంచవ్యాప్తంగా ఒకే శ్లోకం.
        ◀▶
        నావిగేషన్ & కీబోర్డు
        ← → బటన్లు లేదా Arrow Keys తో శ్లోకాలు మారవచ్చు. R = Random శ్లోకం. Space / Enter = Reveal (అవధానం టాబ్ లో). ↺ బటన్ = మళ్ళీ Random.
        🕐
        చివరిగా చదివిన శ్లోకం
        మీరు చివరిగా చదివిన శ్లోకం తదుపరిసారి యాప్ తెరిచినప్పుడు స్వయంచాలకంగా వస్తుంది — అధ్యాయం అర్ధంలో ఆగిపోయినా, పునఃప్రారంభించవచ్చు.
        భగవద్గీత గురించి (BG Meta Panel)
        ℹ️
        Meta Panel తెరవడం
        గీత ఎంచుకున్నప్పుడు పైన "శ్రీమద్భగవద్గీతా ›" కార్డ్ వస్తుంది. నొక్కితే విస్తరిస్తుంది. లోపల అనేక విభాగాలు — ఒక్కొక్కటి నొక్కి చదవండి.
        👤
        కృష్ణ & అర్జున నామాలు
        నామాల విభాగంలో కృష్ణుడి 14, అర్జునుడి 16 పేర్లు వివరాలతో ఉంటాయి. ప్రతి పేరు పక్కన ఆ పేరు ఉన్న శ్లోక సంఖ్యలు చిన్న chip లుగా ఉంటాయి — నొక్కితే నేరుగా ఆ శ్లోకానికి వెళ్తారు. శ్లోకంలో ఆ పేరు amber రంగులో highlight అవుతుంది.
        ↩️
        నామ chip నుండి వెనక్కి రావడం
        శ్లోకానికి వెళ్ళాక పైన "← నామాల జాబితా" బటన్ కనిపిస్తుంది — నొక్కితే మళ్ళీ నామాల విభాగానికి తిరిగి వస్తారు.
        🎬
        తెలుగు సినిమాలో గీత
        అపరిచితుడు (18.66), సాహో (2.54) వంటి చిత్రాలలో ఉపయోగించిన శ్లోకాలు chip లుగా ఉంటాయి — నొక్కితే ఆ శ్లోకానికి వెళ్తారు.
        అవధానం (Practice)
        👁
        పరీక్ష విధానాలు
        పాద 1–4: చూపించిన పాదం చూసి మిగిలినవి చెప్పండి.
        శ్లో#: శ్లోక సంఖ్య చూసి పాఠం చెప్పండి.
        అక్షరం: మొదటి అక్షరం చూసి పాఠం చెప్పండి.
        ఉవాచ: వక్త (కృష్ణ/అర్జున/సంజయ) చూసి పాఠం చెప్పండి.
        1వ/చివరి: అధ్యాయంలో మొదటి/చివరి శ్లోకం అభ్యసించండి.
        📝
        పద×పదం అర్థం (WBW)
        Word-by-word అర్థం English మాత్రమే లభ్యం. అర్థ భాషను En కి మార్చండి. Settings → Meaning Language → En.
        ⌨️
        కీబోర్డు సత్వరమార్గాలు
        Space / Enter = Reveal. ← → = మునుపటి/తదుపరి శ్లోకం. R = Random శ్లోకం.
        వెతుకు (Search)
        🔍
        శ్లోక సంఖ్య వెతుకు
        2.54 లేదా 2:54 టైప్ చేస్తే గీత 2వ అధ్యాయం 54వ శ్లోకానికి నేరుగా వెళ్తారు.
        🔢
        Filter-aware సంఖ్య వెతుకు
        Filter ఎంపిక చేసి సంఖ్య టైప్ చేయండి:
        గీతా + 42 → అన్ని అధ్యాయాల 42వ శ్లోకాలు చూపిస్తుంది.
        VSN + 42 → 42వ నామం + 42వ శ్లోకం (≤108 అయితే రెండూ).
        #42 = VSN 42వ నామం (ఎప్పుడూ).
        ch2 = గీత 2వ అధ్యాయం మొత్తం.
        📝
        పాఠ్య వెతుకు
        నామం లేదా శ్లోక పాఠ్యం Telugu/IAST/Sanskrit లో వెతకవచ్చు. diacritics (ā, ī, ū, ṣ…) లేకుండా సాధారణ అక్షరాలతో కూడా మ్యాచ్ అవుతుంది — krsna → Kṛṣṇa.
        శ్లోక సంఖ్య & రంగులు
        🏷
        Badge రంగులు
        నీలం = అధ్యాయ సంఖ్య (Ch). పచ్చ = శ్లోక సంఖ్య (S#).
        వక్త badges: నీలం = శ్రీ భగవానువాచ · నారింజ = అర్జున ఉవాచ · ఊదా = సంజయ ఉవాచ · గులాబీ = ధృతరాష్ట్ర ఉవాచ.
        ✨
        నామ highlight
        నామాల chip నొక్కి శ్లోకానికి వెళ్ళినప్పుడు, ఆ పేరు ఉన్న పాదం 2.5 సెకన్లు amber రంగులో మెరుస్తుంది — వెంటనే గుర్తుపట్టవచ్చు.
        శ్రీవిష్ణుసహస్రనామమ్
        🌟
        నక్షత్ర సమాచారం
        VSN లో నక్షత్ర badge నొక్కితే ఆ నక్షత్రం గురించి వివరాలు — రాశి, నక్షత్ర గ్రహం, దేవత — చూడవచ్చు.
        🔢
        నామ సంఖ్య vs శ్లోక సంఖ్య
        VSN కి 1000 నామాలు, 108 శ్లోకాలు ఉన్నాయి. Search లో VSN filter తో 42 టైప్ చేస్తే నామ #42 + శ్లోక #42 రెండూ వస్తాయి. #42 అంటే నిర్దిష్టంగా నామ #42 మాత్రమే.
        📋
        VSN Meta Panel
        VSN ఎంచుకున్నప్పుడు "శ్రీవిష్ణుసహస్రనామమ్ ›" కార్డ్ వస్తుంది — గ్రంథం గురించి, ఎవరు చెప్పారు, ఎందుకు పఠించాలి, ఫలశ్రుతి వివరాలు చూడవచ్చు.

        Script & Language
        🔤
        Script selector
        Switch between Telugu (తె), Devanāgarī (सं), and Roman IAST script using the pills in the Reader tab or in Settings. Your choice persists across sessions.
        🎨
        Lipi colour vs meaning colour
        Accent colour (maroon by default) = script text — verse padas, names. Standard colour = meanings, labels, UI. This makes it immediately clear which text changes when you switch script.
        🌐
        UI Language vs Meaning Language
        UI Language — buttons and labels in Telugu or English. Meaning Language — choose which language verse meanings appear in: English (En), Telugu (తె), or Sanskrit (सं).
        Reader tab
        📚
        Text selection
        Choose Śrīmad Bhagavadgītā or Śrī Viṣṇu Sahasranāmam from the dropdown in the Reader tab. Your selection is remembered.
        📖
        Chapter selection
        Tap one or more chapter buttons to select them. With multiple chapters selected, ↺ Random picks from all of them. Tapping a chapter immediately shows a verse from it.
        ☀️
        Today's Verse
        The ☀️ button shows a verse assigned to today's date — cycles through all 700 Gita shlokas, same verse worldwide for the same date.
        ◀▶
        Navigation & keyboard
        Use ← → buttons or keyboard Arrow Keys to move between verses. R = random verse. ↺ = another random from selected chapters.
        🕐
        Last visited verse
        The app remembers the last verse you read. When you reopen it, you resume exactly where you left off.
        BG Meta Panel
        ℹ️
        Opening the panel
        When Gita is selected, a "Śrīmad Bhagavadgītā ›" card appears at the top. Tap it to expand. Inside are several collapsible sections — tap each to explore.
        👤
        Names of Kṛṣṇa & Arjuna
        The Names section lists 14 names of Kṛṣṇa and 16 of Arjuna with meanings. Each name has verse number chips — tap any chip to jump directly to that verse. The name is highlighted in amber for 2.5 seconds.
        ↩️
        Back from name chip
        After jumping to a verse via a name chip, a "← Names list" button appears at the top — tap it to return to the names section.
        🎬
        Gita in Telugu Cinema
        Film cards for Aparichitudu (18.66) and Saaho (2.54) have verse chips — tap to jump to the exact verse used in that film.
        Avadhānam (Practice)
        👁
        Test modes
        Pada 1–4: given a pada, recall the rest.
        Verse #: given the number, recite the verse.
        First letter: given the opening syllable, recite the verse.
        Uvāca: given the speaker, recite the verse.
        First/Last: practise the opening/closing verse of each chapter.
        📝
        Word-by-word meaning (WBW)
        Available in English only. Set Meaning Language to En in Settings to see it.
        ⌨️
        Keyboard shortcuts
        Space / Enter = Reveal. ← → = previous / next verse. R = random verse.
        Search tab
        🔍
        Verse number search
        Type 2.54 or 2:54 to jump directly to Gita chapter 2, verse 54.
        🔢
        Filter-aware number search
        Select a filter pill first, then type a number:
        Gita + 42 → all chapters' verse 42 (up to 17 results).
        VSN + 42 → name #42 + shloka 42 (if ≤108, two results).
        #42 = always VSN name #42.
        ch2 = all verses of Gita chapter 2.
        📝
        Text search
        Search names or verse content in Telugu, IAST, or Sanskrit. Diacritics are optional — typing krsna matches Kṛṣṇa, partha matches Pārtha.
        Verse badges & colours
        🏷
        Badge colours
        Blue = chapter number (Ch). Green = shloka number (S#).
        Speaker badges: Blue = Bhagavān · Orange = Arjuna · Purple = Sañjaya · Pink = Dhṛtarāṣṭra.
